--- layout: main title: skuid.utils --- skuid.utils Object ================= The skuid.utils object provides convenience methods for commonly used functionality in Skuid. ## METHODS ##### `decodeHTML( value )` Decodes html characters like &amp; into their actual character. ##### `encodeHTML( value )` Encodes html characters like & into their encoded value like &amp;. ##### `registerPlugin( name, methods )` Registers a jQuery plugin in the standard skuid component format. {% highlight javascript %} var methods = { init : function( xmlDef ) { // Build the component } }; skuid.utils.registerPlugin('pluginName',methods); {% endhighlight %}
