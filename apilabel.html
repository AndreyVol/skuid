--- layout: main title: skuid.label --- skuid.label Object ================= ### METHODS: #### skuid.label.parse( mergeString, leftSide, rightSide ) Parses a string @mergeString that contains merges delimited on the left by @leftSide (e.g. '{{') and on the right side by @rightSide (e.g. '}}') and returns two arrays: the first containing the merges (the replacement values) and the second containing the formats (text in between merges). #### skuid.label.read( labelName, defaultValue ) Returns the Custom Label named @labelName, (optionally) returning @defaultValue if no such label exists
