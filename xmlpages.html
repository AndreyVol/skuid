--- layout: main title: Skuid Developer Guide --- Collection of standard pages ================= This is a repository of xml files - which provide the instruction to Skuid to correctly build a custom page. ## Instructions The xml files below form the basis for Skuid pages - instructing the application how to build the page. The XML files can be copied and pasted into a new page in the page builder. Find a detailed, step-by-step [tutorial of this process](http://help.skuidify.com/s/tutorials/m/supercharge-your-ui/l/102530-share-a-skuid-page-between-orgs) in our help files. ## Pages ### Home Page + [Customized Home Page](https://github.com/skuidify/skuid/blob/master/pages/CustomHomePage) ### Account Object + [Account Tab Starter Page](https://github.com/skuidify/skuid/blob/master/pages/Account%20Tab%20Starter%20Page) + [Account Detail Starter Page](https://github.com/skuidify/skuid/blob/master/pages/Account%20View%20Starter%20Page) ###Case Object + [Case Management Queue Page](https://github.com/skuidify/skuid/blob/master/pages/Case%20Management%20Page) ###Contact Object + [Contact Detail Starter Page](https://github.com/skuidify/skuid/blob/master/pages/Contact%20Detail%20Starter%20Page) + [Contact Tab Starter Page](https://github.com/skuidify/skuid/blob/master/pages/Contact%20Tab%20Starter%20Page) + [Contact Management Queue Page](https://github.com/skuidify/skuid/blob/master/pages/ContactManagement) + [New Contact Wizard](https://github.com/skuidify/skuid/blob/master/pages/New%20Contact%20Wizard) ###Lead Object + [Lead Detail Starter Page](https://github.com/skuidify/skuid/blob/master/pages/Lead%20Detail%20Starter%20Page) + [Lead Management Queue Page](https://github.com/skuidify/skuid/blob/master/pages/Lead%20Management%20Page) + [Lead Tab Starter Page](https://github.com/skuidify/skuid/blob/master/pages/Lead%20Tab%20Starter%20Page) ###Opportunity Object + [Opportunity Detail Starter Page](https://github.com/skuidify/skuid/blob/master/pages/Opportunity%20Detail%20Starter%20Page) + [Opportunity Tab Starter Page](https://github.com/skuidify/skuid/blob/master/pages/Opportunity%20Tab%20Starter%20Page) + [Opportunities Report](https://github.com/skuidify/skuid/blob/master/pages/OpportunitiesReport) + [New Opportunity Wizard](https://github.com/skuidify/skuid/blob/master/pages/New%20Opportunity%20Wizard)
